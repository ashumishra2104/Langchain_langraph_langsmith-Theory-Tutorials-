# Blog Agent V1: Basic Parallel Writer

This is the initial version of the **Blog Writer AI Agent**. it demonstrates the foundational concepts of **LangGraph orchestration**, including state management, structured outputs, and basic parallel execution (Fan-Out).

## 💡 Key Concepts
- **Shared State**: Implements a `TypedDict` to track the blog topic, planned sections, and completed content across multiple AI nodes.
- **Structured Output**: Uses Pydantic models to force the LLM to return clean data (lists of titles) rather than conversational text.
- **Basic Parallelism**: Uses the `Send()` decorator to launch exactly 3 section writers at the same time, demonstrating how to speed up agentic workflows.
- **Linear Compilation**: A simple "Compiler" node that collects all parallel outputs and stitches them into a final string.

## 📂 Project Files
- `blog_agent.py`: The core agent logic and graph definition.
- `blog_post.md`: Sample output generated by the agent.
- `diagram.png`: Visual representation of the agent's graph.
- `requirements.txt`: Necessary Python packages.

## 🚀 How It Works
1. **Planner**: Takes the topic and generates exactly 3 section headings.
2. **Fan-Out**: Distributes each heading to an independent writer.
3. **Writers**: Each writer produces 2-3 paragraphs for their assigned heading.
4. **Compiler**: Joins the sections and prepares the final text.
