# Blog Writer Agent V4 - AI Image Edition

🚀 **Build a state-of-the-art Blog Writer Agent with AI-generated images and a professional Streamlit UI.**

This version of the Blog Agent leverages LangGraph for complex workflows, OpenAI for content orchestration, Tavily for real-time research, and Google Gemini 2.5 Flash for high-fidelity technical image generation.

## ✦ What's New in V4 ✦

- **AI-Generated Images:** Real content-aware diagrams and visuals embedded directly in the blog.
- **Full Streamlit UI:** A 5-tab web interface for non-technical users to generate, preview, and download blogs.
- **3-Node Reducer Subgraph:** A dedicated pipeline for merging content, deciding on image placement, and generating visuals.
- **Temporal Research:** Date-aware searches that filter evidence by recency.
- **Graceful Fallbacks:** Production-ready error handling for failed image generations.

## 🛠 Project Structure

- `bwa_backend.py`: The core LangGraph logic, data models, and agent nodes.
- `bwa_frontend.py`: The Streamlit web interface and real-time streaming UI.
- `samples/`: Example blog outputs generated by the agent.

## 🚀 Getting Started

```bash
pip install -r requirements.txt
streamlit run bwa_frontend.py
```

Create a `.env` file with:
- `OPENAI_API_KEY`
- `TAVILY_API_KEY`
- `GOOGLE_API_KEY`
