# Blog Writer Agent V4 - AI Image Edition

🚀 **Build a state-of-the-art Blog Writer Agent with AI-generated images and a professional Streamlit UI.**

This version of the Blog Agent leverages LangGraph for complex workflows, OpenAI for content orchestration, Tavily for real-time research, and Google Gemini 2.5 Flash for high-fidelity technical image generation.

---

## ✦ What's New in V4 ✦

V4 adds major capabilities over previous versions:

- **AI-Generated Images:** Real content-aware diagrams and visuals embedded directly in the blog.
- **Full Streamlit UI:** A 5-tab web interface for non-technical users to generate, preview, and download blogs.
- **3-Node Reducer Subgraph:** A dedicated pipeline for merging content, deciding on image placement, and generating visuals.
- **Temporal Research:** Date-aware searches that filter evidence by recency (e.g., last 7 days for news roundups).
- **Graceful Fallbacks:** Production-ready error handling for failed image generations.

---

## 🛠 Project Structure

- `bwa_backend.py`: The core LangGraph logic, data models, and agent nodes.
- `bwa_frontend.py`: The Streamlit web interface and real-time streaming UI.
- `images/`: Local storage for AI-generated visual assets.
- `samples/`: Example blog outputs generated by the agent.

## 🚀 Getting Started

### 1. Prerequisites

Install the required Python packages:

```bash
pip install langgraph langchain-openai langchain-community pydantic python-dotenv google-genai streamlit pandas
```

### 2. API Keys

Create a `.env` file in the root directory with the following keys:

- `OPENAI_API_KEY`: For the core reasoning and writing.
- `TAVILY_API_KEY`: For real-time web research.
- `GOOGLE_API_KEY`: For Gemini 2.5 Flash image generation.

### 3. Running the App

Launch the Streamlit interface:

```bash
streamlit run bwa_frontend.py
```

---

## 🧠 How Image Generation Works

The Agent doesn't just generate text; it acts as an **Expert Technical Editor**:

1. **merge_content:** Sorts and joins parallel worker outputs into a coherent blog.
2. **decide_images:** Analyzes the full blog to decide exactly where diagrams or flows would improve understanding (Max 3).
3. **generate_and_place_images:** Calls Gemini 2.5 Flash to generate unique PNGs, saves them, and embeds them into the Markdown.
